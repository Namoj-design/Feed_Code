# Changelog

## [0.1.0] - 2026-01-29

### Added
- Initial SDK implementation with 18 event types
- Privacy-first architecture with PII filtering
- Session lifecycle tracking
- Navigation and interaction monitoring
- Performance metrics collection
- Python intelligence layer with AI-powered insights
- Friction pattern detection (4 types)
- LLM-based intent inference
- FastAPI backend for event ingestion
- Demo e-commerce application
- Comprehensive documentation

### Security
- Automatic PII redaction
- Anonymous session IDs
- Client-side data filtering
- Opt-out mechanism
